# 教育知识库 RAG 系统 — 分阶段实施计划

## Context
基于《需求说明.md》"项目目标与功能范围"，从零构建一个面向教育培训场景的智能知识库系统。支持内容导入、课程检索、文档检索、题库检索、知识问答和流式交互。

## 技术栈

| 层次 | 技术选型 | 说明 |
|------|---------|------|
| Web 框架 | **FastAPI** | 异步支持，自动生成 OpenAPI 文档，原生 SSE 流式输出 |
| RAG 框架 | **LangChain** | 文档加载、分割、检索链、Prompt 管理 |
| 向量数据库 | **ChromaDB** | 轻量级嵌入运行，适合本地开发和中小规模场景 |
| 嵌入模型 | **BGE-M3** (sentence-transformers) | 本地 GPU 运行，1024 维向量，支持中英文 |
| LLM | **OpenAI 兼容 API** | 对接 SiliconFlow (qwen-flash)，可切换其他后端 |
| 元数据存储 | **SQLite + SQLAlchemy** | 课程、题库、导入任务、对话等结构化数据 |
| 数据校验 | **Pydantic v2** | 请求/响应模型、Settings 配置管理 |
| 异步任务 | **BackgroundTasks** | 导入任务异步执行与状态追踪 |
| 文档解析 | **PyPDF2 + python-docx** | 支持 PDF、Word、Markdown、TXT |
| 重排序 | **BGE-Reranker** | 可选，提升检索精度 |

> **配置管理**: 所有 API Key、LLM Base URL、模型路径、设备等敏感或可变配置统一通过 `.env` 文件管理。提供 `.env.example` 模板，`.env` 加入 `.gitignore`。

## 启动与验证

```bash
# 安装依赖
uv sync

# 复制配置文件，填入实际值
cp .env.example .env

# 启动服务
uv run uvicorn app.main:app --host 0.0.0.0 --port 8000

# Swagger UI
http://127.0.0.1:8000/docs

# 运行测试
uv run pytest tests/ -v
```
